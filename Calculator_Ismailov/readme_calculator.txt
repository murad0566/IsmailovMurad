Привет! Для использования калькулятора необходимо запустить скрипт Calculator_Ismailov.py и далее
передать необходимый аргумент для вычисления в следующем виде: 'yourparams'

Пример ввода: python3 Calculator_Ismailov.py '(2+ -31) ; 20'

Программа состоит из трех основных функций-шагов: 1 - подготовка строки к расчетам (разбиение на операции и числа с учетом унарного минуса)
2 - определение операции, которые вызываются относительно их приоритетности далее на основе словаря с заданными их приоритетами
3 - калькулирование подготовленной строки с помощью приоритезированных операций из функции выше
Пользуйтесь!))
